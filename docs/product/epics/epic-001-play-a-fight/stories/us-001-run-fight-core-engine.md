# 🥇 US-001: Run Fight (Core Engine)

## Story
System executes a fight between submitted codes in a sandbox.

## Why
This is the core of the entire platform - nothing else matters without it.

## Scope
- Execute 2+ agents
- Sandbox execution (time/memory limits)
- Deterministic outcome
- Generate raw logs

## Done when
- A fight can run end-to-end without crashing
- Result + logs are produced
